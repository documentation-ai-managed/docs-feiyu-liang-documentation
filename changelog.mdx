## Upgrading Your Instance

Follow these steps to upgrade to the latest version.

<Steps>
  <Step title="Backup Data" icon="database">
    Export your documentation using the built-in tool:

    ```bash
    feiyu-docs export --output ./backup
    ```
  </Step>
  <Step title="Update Package" icon="download">
    Install the new version:

    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm install feiyu-docs@latest
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```bash
        yarn add feiyu-docs@latest
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Run Migrations" icon="settings">
    Apply database changes:

    ```bash
    feiyu-docs migrate
    ```
  </Step>
  <Step title="Restart Server" icon="refresh-cw">
    Restart your documentation server.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Example: Embed documentation widget
  import { FeiyuDocsWidget } from 'feiyu-docs';

  const widget = new FeiyuDocsWidget({
    apiKey: 'YOUR_API_KEY',
    theme: 'blue' // Matches brand color #3B82F6
  });
  widget.render('#docs-container');
  ```
  ```python
  # Example: Python client integration
  from feiyu_docs import Client

  client = Client(api_key='YOUR_API_KEY')
  docs = client.get_docs(space_id='your-space-id')
  print(docs.title)
  ```
